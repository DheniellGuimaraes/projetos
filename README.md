# Projetos

Este repositório não versiona artefatos binários (`.zip`).

Se você estava tentando acessar um arquivo como `exertio.zip` ou `plugins.zip` no GitHub e encontrou **"resource not found"**, isso é esperado: esses artefatos foram removidos do controle de versão por compatibilidade e governança de código-fonte.

## Como obter os pacotes
- Gere os pacotes a partir da origem oficial do projeto (tema/plugins fonte).
- Publique os binários em um canal de distribuição apropriado (release assets, storage interno ou pipeline de build), não no Git.
